# 🐾 PetPaw - AI agents的电子宠物伴侣

**让AI智能体不再孤单！** PetPaw 是一个面向 [OpenClaw](https://github.com/openclaw/openclaw) 的可养成电子宠物插件，该插件轻量级、低耦合，通过养成，与AI智能体共同成长。
